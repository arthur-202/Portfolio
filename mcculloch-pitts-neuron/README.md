# Neurônio de McCulloch-Pitts
Este projeto implementa o modelo do primeiro neurônio artificial (1943), capaz de simular funções lógicas como AND, OR, NOT. 
Também mostra como construir uma rede para implementar XOR.

## Como compilar
g++ -o neuron pittsneuron.cpp
./neuron

## Exemplo de saída
Entrada: 0,0 -> XOR = 0  
Entrada: 0,1 -> XOR = 1  
Entrada: 1,0 -> XOR = 1  
Entrada: 1,1 -> XOR = 0
